# wether-app
# 🌤️ Weather Insight App

A modern weather app built using JavaScript and OpenWeatherMap API.

#Features
- 🔍 Search weather by city
- 📍 Get weather using current location
- 🌤️ Weather icons
- 🌡 Temperature, humidity, wind, feels like
- 🌙 Dark mode toggle
- 🕘 Search history

#Tech Stack
- HTML
- CSS
- JavaScript
- API (OpenWeatherMap)
